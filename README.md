Projeto feito em .Net, utilizando o LLM Ollam chamando o ToolCalling de forma a interagir com ferramentas externas.

A chamada de ferramenta é uma funcionalidade avançada dos modelos de linguagem (LLMs) que permite que eles interajam 
com ferramentas externas, APIs, bancos de dados ou funções personalizadas para realizar tarefas além do seu conhecimento pré-treinado.
